Smart Pricing Table Builder - Documentation

Thank you for purchasing Smart Pricing Table Builder! This documentation will guide you through installation, usage, and customization of the plugin.

Installation

  1. Download the plugin ZIP file.
  2. Go to your WordPress Dashboard → Plugins → Add New → Upload Plugin.
  3. Select the ZIP file and click Install Now.
  4. Activate the plugin after installation.

Usage

  1. After activation, create or edit any Page or Post.
  2. Insert the shortcode: [smart_pricing_table]
  3. Publish the page and view your new pricing table!

Features Overview

  • Responsive Design: Looks great on all devices.
  • Monthly/Yearly Toggle: Switch prices dynamically.
  • Dark Mode: Toggle between light and dark themes.
  • Export to PDF: Download the table as a PDF.
  • Copy HTML: Copy the table's HTML to the clipboard.
  • Modern UI: Gradient buttons, animations, FontAwesome icons.

Customization

To customize your pricing plans:

  1. Open the plugin folder → templates/pricing-table.html.
  2. Edit the Plan Titles, Prices, and Feature Lists.
  3. Save the file and refresh your WordPress page to see changes.

FAQs

Can I add more plans?

Yes! Just duplicate the <div class="sptb-plan">...</div> block inside pricing-table.html.

How do I change the colors?

Open style.css and modify the gradient backgrounds under .sptb-toggle button and .sptb-button.

Does it require any API keys?

No, the plugin works 100% standalone without any external API keys.
