Build beautiful, responsive pricing tables for WordPress with monthly/yearly toggles, tooltips, dark mode, and PDF export — no coding needed!

== Description ==

Smart Pricing Table Builder is a lightweight, beginner-friendly WordPress plugin that helps you create modern, responsive pricing tables effortlessly. Add toggles between Monthly and Yearly pricing, allow users to export tables as PDF, switch between Light/Dark Mode, and more — all with a simple shortcode.

= Core Features =
* Easy shortcode [smart_pricing_table]
* Responsive pricing tables
* Monthly/Yearly toggle switch
* Tooltips and icons on features
* Dark Mode toggle support
* Export pricing table as PDF
* Copy HTML to clipboard
* Modern gradient buttons and animations
* Lightweight and fast
* No coding skills required!

= How to Use =
1. Install and activate the plugin.
2. Create a new page or post.
3. Insert the shortcode `[smart_pricing_table]` where you want the table to appear.
4. Customize your plans by editing the pricing-table.html template.

= Installation Instructions =
1. Upload the plugin files to the `/wp-content/plugins/smart-pricing-table/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the shortcode `[smart_pricing_table]` on any page or post.

= Frequently Asked Questions =

**Q: Can I customize the plans and features?**  
A: Yes! Simply edit the `templates/pricing-table.html` file to add your own plans, prices, and features.

**Q: Does it support Dark Mode?**  
A: Yes, it includes a built-in Dark Mode toggle button.

**Q: Can I export the pricing table?**  
A: Absolutely! You can export the table as a downloadable PDF file or copy its HTML to the clipboard.
